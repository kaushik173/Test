Data Entry - Action Request

Creating an action request consists of two pages, a List and an Add page.

Action Request List Page: The list page displays all action requests that have been created on the case and is sorted by date in ascending order.  To create an action request, click either the "Add Action Request" or "Add Standard Action Request" link, which routes to the Action Request Add page (see below).

To print an action request, simply click the Print link (left-most column) on the desired request.  Upon clicking the print link, the printable version is displayed in another window.

Action Request Add Page: Upon coming to this page the following fields will have a default value:

·        Request Date (today’s date)

·        Request Type (Rpt/Inv)

·        Request By (current attorney)

·        Tasks (the standard tasks, only if “Add Standard” was used)

Besides the fields above, you are also required to enter a Due Date, select a Request For staff member, at least one client, at least one task, and enter a Request Note in order to submit the request.  Upon clicking the Submit button you are routed to the Action Request List where the newly submitted request will display.
