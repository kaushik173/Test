Tasks - Quick Add Case

The Quick Add Case task is a two step process.  First, the user must enter the case number(s) so Jcats can validate whether or not the number(s) already exists on a case.  If the number(s) already exists then the user is routed to the Main page of the existing case, so they can begin to update the case.  Second, if the case number(s) do not exist then the user is routed to the Quick Add Case page where they can quickly enter the following case information on a one-page data entry page:

  • Roles (up to three children and three adults)
  • Clients (singular if an adult, multiple if children)
  • Associations
  • Address and “lives with” relationships
  • Legal Parties (judge and inner-office attorney)
  • Court Department and Initial Hearing
  • Petition Type and Allegations
  • Note: initial case note
  • Language: Spanish speaking checkbox for each role
