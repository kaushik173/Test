Data Entry - Notes


Notes constists of two pages, a List and an Add/Edit page.

List Page: The list page displays all notes added on the case and is sorted by date in descending order.  The subject and "Add Note" links are used for updating and adding notes respectively.

To view a printable version of the notes, simply check the desired records and click the "Print" link at the top of column one.  The checkbox in the column header is the toggle switch used to check all and uncheck all of the checkboxes.

Add/Edit Page: Select the Note Type and enter the Subject and the Note.  If you want the note broadcasted on a particular page (i.e. Main Page) then check the checkbox for the desired page.  When a page has broacast notes then the note icon displays in the status bar to the right of the JCATS #.  When clicking on that icon a popup appears and the user is able to view all notes broadcasted to that page.

Note: Notes can only be edited by the user that entered them.
