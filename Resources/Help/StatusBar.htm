Status Bar

The status bar is found at the top of every Jcats page.  It summarizes the active case and allows you to quickly route back to the Main Page by clicking on the JCATS# link.  The fields are as follows:

 

Field

 Description

 Case Name

 role selected by user to represent the case name

 Case #

 petition number associated to the case name

 Appt. Date

 Case appointment date

 Status

 open or closed

 Attorney

 current attorney

 JCATS #

 this is a link that routes to the Main Page of the case

 

Additional Status Bar item:

Note Icon: If a case note is associated to a Data Entry page then a note icon will display to the right of the JCATS # field.  Upon clicking on this icon a popup displays all notes that were broadcasted to the Data Entry page.
