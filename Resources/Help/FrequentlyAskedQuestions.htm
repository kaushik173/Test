Frequently Asked Questions


How do I add a new case? Quick Add Case

How do I record time? Record Time

How do I create a subpoena? Subpoena

How do I close a case?  On the Hearing Edit page, a petition is closed when a “Closure” result is entered for the petition.  Closure codes all begin with the letters “CLSD”.  The Jcats case (sibling group) is automatically closed when all petitions on the case are closed.  If you want to close a case where a hearing does not exist then add a hearing of type “Administrative Closure” and then enter the “Closure” result.

How do I reopen a closed case? If a new petition is involved then go to the Main page and click the Add Petition link.  If the case closed because an incorrect hearing result was entered then go to the Hearing Edit page (for that hearing) and change the result.  To do this, select the blank result and then use the Global Result dropdown to select the correct result.

How do I reset a user account that has been locked?

  • Go to Administration - Users and search for the account, being sure to uncheck the “Active Users” checkbox.
  • Click on the Login Name, which routes you to the User Edit page.
  • On the User Edit page, remove the user end date, reset the password and click the Save button.
