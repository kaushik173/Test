Data Entry - Contact Information

This page consists of the following two sections:

Edit Contact Information (top section): This section displays a list of all contact numbers/addresses that have been added to the case.  If the contact information is no longer valid then it can be deleted by using the "Del" hyperlink.

Add Contact Information (bottom section): This section allows you to add up to five different contact number/address at a time.  Upon clicking the Process button the page redisplays with the newly added information in the top section, and allows you to enter more numbers if needed.
