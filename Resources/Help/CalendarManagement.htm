Inquiry - Calendar Management

This page consists of two sections, Date Range and Search Results.

Date Range: The top and bottom calendars are used for the start and end date search criteria respectively.  The default range is set to today, but can easily be modified by using the numeric links within either of the calendars.

Search Results: The Process button will return the list of attorneys that have hearings scheduled during the date range, sorted by attorney name.  The Attorney link routes to the My Calendar page, which is the page each individual attorney uses to view and print their own calendar.

Printable Calendar: the Print button opens up a new window in order to display the printable version of the calendar with a page break per attorney.
