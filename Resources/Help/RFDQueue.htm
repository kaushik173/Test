Tasks - My AR Queue

This page consists of two sections, Search Criteria and Search Results.  It includes all action requests ever "submitted by" or "requested for" the staff member.

Search Criteria: The Start Date (default = today) and End Date criteria are dependent upon the Date Range Type that is selected (default = "Due Date").  Leave the Start Date field blank to return any past due requests.

Search Results: The Search/Process button will return the list of action requests that match the search criteria, sorted by Due Date.  The Type link routes to the AR Edit page, which is part of a submenu that includes the ability to enter/update addresses, profile questionnaires, and time worked.

Buttons: The Transfer button routes to a page that allows supervisors to transfer action requests from one staff member to another.  The Print button opens up a new window in order to display the search results in a report format.
