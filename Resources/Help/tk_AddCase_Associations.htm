Data Entry - Associations

This page allows you to associate one person to another and consists of the following two sections:

Add Associations (top section): select a Person and an Association type from the dropdowns and then check the applicable names in the Related Person multi-select.  Upon clicking the Add button the page will redisplay with the newly added associations in the Association in Case section.

Associations in Case (bottom section):  This section lists all associations that have ever been entered on this page.  For historical purposes, each record has a start date and an end date field.  Use the "Del" hyperlink to delete an association that has been entered in error.
