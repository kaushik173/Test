Data Entry - Addresses and Placements

This page constists of the following two sections:

Address Entry (top section): when placing a person at an address or placement facility, you need to do one and only one of the following:

·        Freeform Enrty: Street, City, State, Zip, County and Phone

·        OR Existing Address (selection of existing addresses on case)

·        OR Placement Agency/Institution (dropdown select)

People In Case (bottom section):  When using one of the above methods, you must select the type of address from the "Type" dropdown for the respective people.  For instance, if a caretaker moves and a new address is entered in the top section (for the caretaker) then you must choose the appropriate address type for the caretatker and the children that now live at the new address with the caretaker.  When doing this, the previous address for each person will automatically receive and end date, thus maintaining a history of the person's addresses.

Address Edit: To edit an address, click the adress hyperlink.  You will be routed to the Address Edit page.  This page should only be used when an address has been incorrectly entered or the phone number has changed.  This should not be used if the person has moved to another address, because overwriting the address would lose the fact that they ever lived at the previous address.

Print: Use the Save/Print button to view a printable verison of all the addresses/placements for the roles on the case.

!! Important: if a new caretaker is introduced to the case, and they do not already play a role on the case, then you should use the Add Legal Parties To Case page.  This way, you can add the caretaker and at the same time indicate which children live with them and how they are related/assocated.
