Inquiry - My Caseload

This page constists of the following two sections:

Search Criteria (top section): use the top section to filter your caseload search result.  To return your entire open caseload, simply select the Case Status of "Open" and leave the other fields blank.

Search Results (bottom section):  The result set is based on the search criteria (in the top section), and is sorted in order of JCATS #.  Use the JCATS # hyperlink to route to the Main Page.  Use the Transfer hyperlink to route to the Transfer Case page.

Print: Use the Print button to view a printable verison of the search result.  The Print Sort criteria (at the top of the page) will determine the order in which the cases appear on the printable version.
