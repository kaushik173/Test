System Short-cuts

There are several shortcuts that users can use to save time in data entry and system navigation.  They are as follows:

Action

Key-Stroke

Refresh page

F5

Back, Refresh page, and Log Out

Right click on screen (not on a field or link)

Today’s Date

Ctrl + D or up/down arrow keys.  Also, use the up/down arrow keys to increase/decrease days.

Yellow square process button

Enter (does not work if cursor is in a note field)

Copy highlighted text to clipboard

Ctrl + C

Paste clipboard text

Ctrl + V

Open additional JCATS browser window

Ctrl + N

Buttons (i.e. Save/Continue)

Alt + underlined letter


There are also convenient Ergonomic Tools:

·        The Plus (+) Icon found on the bottom of certain Pages will increase the font size.

·        The minus (-) Icon will return the font size to its original condition.
