Data Entry - Record Time List (and Add)


The list page displays all time recorded on the case.  You can filter the display by using the Worker and/or Client dropdowns.  The list is sorted by date descending.  The description column provides a link to the edit page in case the record needs to be updated.  You can print all or any number of records by checking the boxes in the first column and then clicking the Print link at the top of that column.
