Data Entry - Subpoena

Creating subpoenas constists of two pages, a List and an Add page.

List Page: The list page displays all subpoenas that have been created on the case and is sorted by date in ascending order.  To create a subpoena, select a hearing from the dropdown and then click the "Add" link, which routes to the Subpoena Add page (see below). 

To print subpoenas, simply check the desired records and click the "Print" link at the top of the checkox column.  The checkbox in the column header is the toggle switch used to check all and uncheck all of the checkboxes.  Upon clicking the print link, the information is merged into a Microsoft Word subpoena template and displayed on the screen.  You can then use your computer to modify/print the document as you would any other Microsoft Word file.

Add Page: The roles on the case that can be subpoenaed are displayed in a list that includes two columns of checkboxes. The first checkbox column is for selecting the role, the second checkbox column is for selecting the address(es).  If only one address exists for the role then it is automatically checked when the role is checked.  Likewise, if the address is checked first then the role is automatically checked.  Upon clicking the Save and Continue button, you are routed to the list page where the newly added subpoenas display.
