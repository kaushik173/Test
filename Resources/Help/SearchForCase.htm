Inquiry - Search for Case

All fields, except Case #, use a telephone book search approach.  As an example, A last name search of 'bar' will return Barr and Barns.  However, it will not return Abare, because although Abare has 'bar' within the name, it starts with the letter 'a'. 

The Case # search is different, because it looks for a match anywhere within the field.  As an example, A Case # search of '2345' will return C-12345A.

You may fill in one or all of the available fields in order to search the database.  Using only the last name criteria is not recommended (unless it is an uncommon name), because it may return a very long list of names.  The more fields that are filled in the more specific the search will be.

The search results are sorted alphabetically.  If the exact same person belongs to multiple Case #'s then you will know it is the exact same person, because all repeated fields will be suppressed. i.e. Last Name, First Name, DOB, Sex.

The JCATS # link routes to the Main Page.  The Close Date link displays a popup that includes the petition type and closure reason.

Note: Use the plus/minus (+/-) icon in the page footer to increase/decrease the font size for your viewing preference.  JCATS remembers your last preference and will continue to use it on all pages that have the viewing preference icon.
